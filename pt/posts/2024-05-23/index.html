Também comecei a colocar em prática tudo o que aprendi, trabalhando no uso de IA para me ajudar com o relatório de estágio. Comecei tentando usar Langchain para pedir ao ChatGPT que resumisse um dos posts do blog. Encontrei este tutorial oficial, então não foi difícil. Você pode verificar o código aqui, mas aqui está o passo a passo (assumindo que você já tenha sua chave no arquivo .env):

  1. Crie um loader para o arquivo que você deseja resumir, no meu caso era um arquivo markdown, então usei UnstructuredMarkdownLoader:
markdown_path = "content/posts/2024-04-16.md"
loader = UnstructuredMarkdownLoader(markdown_path)
  1. Defina o prompt, que é o que você quer que a IA faça (escrever um resumo conciso), o que você vai fornecer a ela (text) e o que você quer em troca (CONCISE SUMMARY):
prompt_template = """Write a concise summary of the following:
"{text}"
CONCISE SUMMARY:"""
prompt = PromptTemplate.from_template(prompt_template)
  1. Defina a LLM chain:
llm = ChatOpenAI(temperature=0, model_name="gpt-4")
llm_chain = LLMChain(llm=llm, prompt=prompt)
  1. Carregue o arquivo, extraia o conteúdo e execute a LLM chain:
docs = loader.load()
content = docs[0].page_content # Você pode verificar como o arquivo markdown está organizado imprimindo `docs`

response = llm_chain.run(content)
print(response)

Quando consegui fazer isso funcionar, comecei a pensar em como obter todos os arquivos da pasta posts para criar um embedding deles. Decidi que só preciso dos arquivos em inglês, já que sempre posso usar o script que fiz para traduzi-los, então aqui está o que acabei fazendo:

import os

folder_path = "content/posts"
files = os.listdir(folder_path) # Retorna uma lista com o nome de cada arquivo no caminho
en_files = [file for file in files if '.pt' not in file and '_index' not in file]

for file in en_files:
    markdown_path = f"{folder_path}{file}"
    loader = UnstructuredMarkdownLoader(markdown_path)

    docs = loader.load()
    content = docs[0].page_content
